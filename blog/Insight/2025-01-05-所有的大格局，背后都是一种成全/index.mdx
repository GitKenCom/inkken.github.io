战国时期，梁、楚两国接壤，两国士卒都在各自边境上种了西瓜。

梁国这边辛勤管理，种出的瓜又大又圆；楚国那边不管不顾，瓜长得又瘦又蔫。

楚国的人又嫉又恨，有天夜里，他们有人偷偷过去把梁国的瓜田毁了。

梁国士卒发现后，非常愤怒，他们对县令宋就说：“我们也去把楚国瓜苗扯断吧！”

宋就听了，笑着说：“冤冤相报，何时能了？楚国的瓜疏于管理才长得不好，我看你们以后不如也帮他们浇浇水吧。”

梁国士卒虽不情愿，还是按宋就的吩咐做了。

很快，在梁国士卒的帮助下，楚国的瓜越长越好。

楚国人知道后，非常羞愧，便把此事上报朝廷，楚王听了也很感动，派人带厚礼送去梁国。

就这样，两国从此结为友好邻邦。

孔子有言：**“己欲立而立人，己欲达而达人。”**

心中只有自己，见不得别人好的人，挡住别人，却也将自己的路堵死。

只有懂得为别人铺路修桥，相互扶持，才能走出一条更宽广的道路。

所有的大格局背后，其实都是一种相互成全。


<div style={{textAlign: 'center', color: '#FF8C00'}}>  
  <h3>1</h3>
  <h3>格局越小的人，越见不得别人好</h3>
</div>


东野圭吾曾说：**“人性最大的恶，是见不得别人好。”**

现实中，越是没本事的人，越是看不惯别人的好，总喜欢拆别人的台。

挖空心思给他人下绊子，最终害了他人，却也坑了自己。

曾看过一个钥匙和锁头的故事。

有把钥匙，觉得自己整天在外颠簸，而锁头则安安稳稳在家，心里很气不过。

这天，钥匙想到个办法，为了赶走锁头，它偷偷将自己藏起来。

主人回家后，找不到钥匙，就把锁头给砸了。

钥匙很开心，从藏身处跳了出来。

但锁已经被主人换了新的，钥匙没了用处，也被主人随手扔掉。

正如有句话所说：“枪响之下，没有赢家。”

相互拆台，终究只会两败俱伤，向他人泼脏水，自己也会被溅得满身是泥。

金牌培训师陈浩，在他的书里讲过这样一个故事。

小青在一家互联网公司做网页设计，工作能力非常强。

有一次，为了让公司的设计外销到新市场，她全心全意投入到方案设计中。

结果，方案比客户约定时间晚了几天，客户对此颇有微词。

虽然最后方案顺利通过了，但领导还是在会议上批评了她。

为此，她心中有些窝火。

自那之后，每当她与客户交流时，经常流露出对领导的不满。

甚至还对客户说，自己的领导不懂设计，有些细节做得不够好，有时为了多接单，还会让她赶工，质量很难保证。

听了她的话，很多客户不再选择他们公司。

后来，公司发现订单量减少，一调查，原来是小青在背后诋毁，便立马辞退了她。

作家水木然曾说，**人之所以痛苦，是因为人与人之间一直在相互为难。**

越是层次低的人，越喜欢给别人种荆棘，最后也让自己寸步难行。

在这世上，所有对他人的刁难与恶意，其实都是给自己挖坑。

**遇事斤斤计较，睚眦必报，未来的路只会越走越窄。**

<div style={{textAlign: 'center', color: '#FF8C00'}}>  
  <h3>2</h3>
  <h3>层次越高的人，越懂得成就他人</h3>
</div>


在书中看过一个故事。

在美国新闻界，《华盛顿邮报》和《华盛顿明星新闻报》是一对死对头。

1972年，《邮报》披露了白宫的一桩丑闻。

总统尼克松非常生气，为了惩罚《邮报》，他表示后续情况只接受《新闻报》的独家采访。

对于《新闻报》来说，这本是一举压过《邮报》的好机会。

但《新闻报》却发表了出乎白宫意料的社论，它表示，假如《邮报》记者不能进入白宫，它们也将停止对该事件的采访报道。

最终，《邮报》和《新闻报》都获得了采访权。

孔子曰：**“君子成人之美，不成人之恶。”**

很多时候，小人只会落井下石，踩着别人往上走。

真正大格局的人，反而懂得相互帮衬，主动善待他人。

上个世纪40年代，戏曲家程砚秋先生曾拜梅兰芳先生为师。

两人本是同行冤家，梅兰芳却对程砚秋倾囊相授。

为了让程砚秋得到更多的学习机会，每场演出前，梅兰芳都要留一张票给程砚秋。

1947年，梅兰芳与程砚秋同时在沪演出，梅兰芳在中国戏院，程砚秋在天蟾舞台。

在戏曲界，两人算是“打擂台”了。

然而梅兰芳却做了出人意料的举动。

他主动询问程砚秋演什么节目，若发现对方安排的节目与自己相同，他就主动更改，绝不与程的重复，以免影响程砚秋的观众数目。

另外，在演出票价上，他也故意不超过程砚秋，以此表明程砚秋的价值和地位。

一段时间，这两个戏院每晚都是人山人海，座无虚席。

在梅兰芳的扶持下，程砚秋也成为一代京剧大师，两人亦师亦友，共同推动了戏曲艺术的蓬勃发展。

清代奇书《幽梦影》中说：**“云映日而成霞，泉挂岩而成瀑。”**

没有太阳照耀，云便无法变成绚烂彩霞；没有山崖托举，泉水便难以成为壮丽的瀑布。

在这世上，人和人之间，只有互相帮助，才能彼此成就。

一个人的能力再强，单打独斗注定无法走远。

**学会为彼此搭台，携手共进，方能在互利互惠中实现共赢。**

<div style={{textAlign: 'center', color: '#FF8C00'}}>  
  <h3>3</h3>
  <h3>成全他人，最后都是成全自己</h3>
</div>

《道德经》里说：“既以为人，己愈有；既以与人，己愈多。”

生活就是这样，你尽力帮助别人，自己会更充实；你尽力给予别人，自己得到的反而可能会更多。

企业家李开复，初到微软公司，资深经理人琼斯经常为难他。

很长时间里，琼斯不仅拒绝李开复参与自己的项目，还对李开复的很多正当要求置若罔闻。

后来李开复凭自身努力，得到一个很大的科研项目。

这时，他却主动邀请琼斯加入。

不久，公司高层听人说，琼斯工作能力有限，人际关系紧张，就准备将其辞退。

当他们向李开复征询意见时，李开复说：

```
琼斯加盟我的团队后，我才有机会近距离观察他。

琼斯气量不大，与人相处有些固执，但我认为，他是个可用之才，工作能力没有问题，应该给他一个机会。

```

因为这句评价，琼斯最终被留用了。

琼斯得知后，真诚向李开复致谢道：“感谢你教会了我怎样做人！”

从此，他便成为李开复最得力的合作伙伴。

在琼斯的帮助下，李开复最终也创建了属于自己的事业。

李开复不念琼斯旧恶，主动对他施以援手，既给对方留下了一方余地，也给自己撑起了一片天空。

想起林清玄在《送一轮明月给他》中的一句话：

```

我们时时保有善良、宽容、明朗的心性，不要说送一轮明月，同时送出许多明月都是可能的。

因为明月不是相送，而是一种相映，能映照出互相的光明。

```

成年人的世界里，没有人能是一座孤岛，可以独立存在。

人与人互相帮忙，为彼此送出自己的月亮，才能照亮各自的前路。

正所谓，成人达己。

力所能及的时候，不妨拉人一把，帮人一下。

**你对别人的善意，最后都会反哺到自身，让自己走得更高更远。**


<div style={{textAlign: 'center', color: '#FF8C00'}}>  
  <h3>▽</h3>
</div>


杨澜曾说过一句话：**“你的活法，就是你的格局。”**

格局越小的人，越以拆台为乐，喜欢拉别人下马，渐渐也把自己的路走死。

格局越大的人，越懂得互利互惠，在守望相助中实现共赢。

就如纪伯伦所说，生命的意义，就在于人与人之间的相互联结。

行走世间，学会相互搭台，彼此成全。
